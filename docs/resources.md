## Essential tools

### Free with your .edu email
- **[Canva Pro](https://www.canva.com/education/)** — Design tool for graphics, presentations, and more
- **[Google Workspace](https://workspace.google.com/)** — Docs, Drive, Slides (2TB storage with MSU account)
- **[Spotify Premium](https://www.spotify.com/us/student/)** — Student discount
- **[GitHub Student Developer Pack](https://education.github.com/pack)** — Free developer tools

### Browser extensions (all free)
- **[uBlock Origin](https://ublockorigin.com/)** — Ad blocker
- **[OneTab](https://www.one-tab.com/)** — Tab management
- **[LastPass](https://www.lastpass.com/)** or **[1Password](https://1password.com/)** — Password manager
- **[Grammarly](https://www.grammarly.com/)** — Writing assistant
- **[Video Speed Controller](https://chrome.google.com/webstore/detail/video-speed-controller/nffaoalbilbmmfgbnbgppjihopabppdk)** — Control video playback speed

### Mac-specific
- **[CopyClip](https://apps.apple.com/us/app/copyclip-clipboard-history/id595191960)** — Clipboard history
- **[Greenshot](https://getgreenshot.org/)** — Screenshot tool (also available for Windows)
